# Gaussian lsp

Language Server Protocol implementation for gaussian quantum chemistry software.

## Features

- Syntax highlighting
- Auto-completion
- Diagnostics
- Hover documentation

## Installation

```bash
pip install gaussian-lsp
```

## Usage

```bash
gaussian-lsp
```

## Development

```bash
git clone https://github.com/newtontech/gaussian-lsp.git
cd gaussian-lsp
pip install -e ".[dev]"
```
